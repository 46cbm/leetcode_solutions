# 用scanf读入不同类型的变量

用scanf输入的各项之间用空格分割，但注意若是读入一个字符，输入空格的话会被当成输入的字符

```c++
#include<iostream>
using namespace std;
int main(){
	int n; char c; float f;
    scanf("%d%c%f", &n, &c, &f);
    printf("%d %c %f", n, c, f);
}
```

输入：

```
34k_234.45
```

输出：

```
34 k 234.449997
```

若输入：

```
34_k_456
```

则会输出：

```
34  0.000000		//第一个空格给了c，k给了f导致f无效
```

```c++
#include<iostream>
using namespace std;
int main(){
	int c;
    while((c = cin.get()) != EOF){		//cin.get()返回int类型，即输入字符的ascii码
        cout << (char)c;
    }
}
```
