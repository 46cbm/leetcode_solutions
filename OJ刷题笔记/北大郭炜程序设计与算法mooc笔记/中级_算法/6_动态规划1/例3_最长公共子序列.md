**思路：**

```
输入两个串s1,s2
设MaxLen(i,j)表示：
	s1的左边i个字符形成的子串，与s2左边j个字符形成的子串的最长公共子序列的长度（i,j从0开始算）
MaxLen(i,j)就是本题的状态

假定len1 = strlen(s1), len2 = strlen(s2)
那么题目就是要求 MaxLen(len1, len2)

显然：（边界条件）
MaxLen(n,0) = 0  (n = 0...len1)
MaxLen(0,n) = 0  (n = 0...len2)

递推公式：
if(s1[i-1] == s2[j-1])
	MaxLen(i,j) = MaxLen(i-1, j-1) + 1;
else
	MaxLen(i,j) = Max(MaxLen(i, j-1), MaxLen(i-1, j));		
//因为第i个和第j个不相同，若相同，则MaxLen(i,j)比MaxLen(i, j-1), MaxLen(i-1, j)都要大。

时间复杂度O(mn),m,n是两个字符串的长度
```



**代码：**

```c++
#include<iostream>
#include<cstring>
#include<algorithm>
using namespace std;

char sz1[1000];
char sz2[1000];

int maxLen[1000][1000];
int main() {
	while (cin >> sz1 >> sz2) {
		int length1 = strlen(sz1);
		int length2 = strlen(sz2);
		int nTmp;
		int i, j;
		for (i = 0; i <= length1; i++)
			maxLen[i][0] = 0;
		for (j = 0; j <= length2; j++)
			maxLen[0][j] = 0;
		
		for (i = 1; i <= length1; i++) {
			for (j = 1; j <= length2; j++) {
				if (sz1[i - 1] == sz2[j - 1])
					maxLen[i][j] = maxLen[i - 1][j - 1] + 1;
				else
					maxLen[i][j] = max(maxLen[i][j - 1], maxLen[i - 1][j]);
			}
		}
		cout << maxLen[length1][length2] << endl;
	}
	return 0;
}
```
