**题目描述：**

```
有12枚硬币（A-L）。其中有11枚真币和1枚假币。假币和真币的重量不同，但不知道谁轻谁重。
现在用一架天平称了这些硬币三次，告诉你结果。请确定谁是假币，并确定假币轻还是重。
天平每边最多放6枚硬币。
（数据保证一定能找出来）
```

**输入：**

```
up表示右边翘起来，即右边轻
even表示一样重
down表示左边翘起来，即左边轻
第一行输入n，表示n组测试数据。
```

**输出：**

```

```

**输入样例：**

```
1
ABCD EFGH even
ABCI EFJK up
ABIJ EFGH even
```

**输出样例：**

```
K is the counterfeit coin and it is light.  	//or heavy
```



**思路：**

```
枚举。
对每一枚硬币，假设它是轻的，看是否符合称量结果。如果符合，问题解决。如果不符合，就假设它是重的，
看是否符合称量结果。把所有硬币都试一遍，一定能找到特殊硬币。
```



**代码：**

```c++
#include<iostream>
#include<string>
using namespace std;

char Left[3][7];
char Right[3][7];
char result[3][7];
bool IsFake(char c, bool light);		//light为真，表示假设硬币轻。否则表示硬币重。

int main() {
	int t;
	cin >> t;
	while (t--) {
		for (int i = 0; i < 3; i++)
			cin >> Left[i] >> Right[i] >> result[i];
		for (char c = 'A'; c <= 'L'; c++) {
			if (IsFake(c, true)) {
				cout << c << " is the counterfeit coin and it is light.\n";
				break;
			}
			else if(IsFake(c, false)){
				cout << c << " is the counterfeit coin and it is heavy.\n";
				break;
			}
		}
	}
	return 0;
}


//返回true:表示假设结果成立
//返回false:表示假设结果不成立
bool IsFake(char c, bool light) {
	for (int i = 0; i < 3; i++) {
		char *pLeft, *pRight;
		if (light) {
			pLeft = Left[i];
			pRight = Right[i];
		}
		else {					//若假设假币重，则左右换一下即可
			pLeft = Right[i];
			pRight = Left[i];
		}
		switch (result[i][0]) {
		case 'u':
			if (strchr(pRight, c) == NULL)	//假币轻，假币应该在右边，假币不在右边，返回false
				return false;
			break;
		case 'e':
			if (strchr(pLeft, c) || strchr(pRight, c))	//假币应该左右都不在
				return false;
			break;
		case 'd':
			if (strchr(pLeft, c) == NULL)	//这时假币应该在左边，若不在，返回false
				return false;
			break;
		}
	}
	return true;
}
```
