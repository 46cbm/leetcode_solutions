# 例3_数制转换

**题目描述：**

```
求任意两个非负整数的转换（二进制~十六进制），所给整数在long所能表达的范围之内。不同进制的符号为(0123456789abcdef)或(0123456789ABCDEF)
```

**输入：**

```
输入只有1行，包含三个整数a,n,b。将a进制数n转换成b进制数输出。a，b是十进制数，2 <= a,b <= 16
```

**输出：**

```
输出占一行，字母用大写
```

**输入样例：**

```
15 Aab3 7
```

**输出样例：**

```
210306
```



**思路：**

```
先转成10进制
```



**代码：**

```c++
#include<stdio.h>
#include<string.h>
using namespace std;

int main() {
	int a, b;
	char str[40];
	while (scanf("%d%s%d", &a, str, &b) != EOF) {
		int tmp = 0, length = strlen(str), c = 1;		//c为初始化权重，最低位0次方为1，之后每升高一位乘a
		for (int i = length - 1; i >= 0; i--) {			//从低位到高位计算
			int x;
			if (str[i] >= '0' && str[i] <= '9')
				x = str[i] - '0';
			else if (str[i] >= 'a' && str[i] <= 'f')
				x = str[i] - 'a' + 10;
			else
				x = str[i] - 'A' + 10;
			tmp += x * c;
			c *= a;
		}
		char ans[40], size = 0;
		do {
			int x = tmp % b;
			ans[size++] = (x < 10) ? x + '0' : x - 10 + 'A';
			tmp /= b;
		} while (tmp);
		for (int i = size - 1; i >= 0; i--)
			printf("%c", ans[i]);
		printf("\n");
	}
	return 0;
}
```
