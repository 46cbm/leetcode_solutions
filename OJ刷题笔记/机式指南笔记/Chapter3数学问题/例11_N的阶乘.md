# 例11_N的阶乘

**题目描述：**

```
输入一个整数N，输出N的阶乘。
```

**输入：**

```
N（0 <= N <= 1000）
```

**输出：**

```
每组一行。
```

**输入样例：**

```
4
5
15
```

**输出样例：**

```
24
120
1307674368000
```



**思路：**

```
输入的数据不大，但是输出的数据很大，需要用高高精度整数。
与上题不同之处有二。一是可以直接读入整数而不是字符串；二是运算符重载乘法。
```



**代码：**

```c++
#include<stdio.h>
#include<string.h>

struct bigInteger{
	int digit[1000];
	int size;
	void Init(){
		for(int i = 0; i < 1000; i++) digit[i] = 0;
		size = 0;
	}
	
	//这题输入数据较小，直接读入的是整数 
	void Set(int x){
		Init();
		do{
			digit[size++] = x % 10000;
			x /= 10000;
		}while(x != 0);
	}
	
	void Output(){
		for(int i = size-1; i >= 0; i--){
			if(i != size-1) printf("%04d", digit[i]);
			else printf("%d", digit[i]);
		}
		printf("\n");
	}
	
	bigInteger operator * (int x) const{
		bigInteger res;
		res.Init();
		int carry = 0;
		for(int i = 0; i < size; i++){
			int tmp = x * digit[i] + carry;
			carry = tmp / 10000;
			tmp %= 10000;
			res.digit[res.size++] = tmp;
		}
		if(carry != 0)
			res.digit[res.size++] = carry;
		return res;
	}
}a; 

int main(){
	int n;
	while(scanf("%d", &n) != EOF){
		a.Init();
		a.Set(1);
		for(int i = 1; i <= n; i++)
			a = a * i;
		a.Output();
	}
	return 0;
}
```
