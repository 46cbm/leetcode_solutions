# 例5_最小公倍数

**题目描述：**

```
给定两个正整数，计算最小公倍数
```

**输入：**

```
多组测试数据，每组一行
```

**输出：**

```
每组一行
```

**输入样例：**

```
10 14
```

**输出样例：**

```
70
```



**思路：**

```
两个数的最小公倍数为两个数的乘积除以它们的最大公约数。
```



**代码：**

```c++
#include<stdio.h>

int gcd(int a, int b) {
	return b == 0 ? a : gcd(b, a%b);
}

int main() {
	int a, b;
	while (scanf("%d%d", &a, &b) != EOF)
		printf("%d\n", a*b / gcd(a, b));
	return 0;
}
```
