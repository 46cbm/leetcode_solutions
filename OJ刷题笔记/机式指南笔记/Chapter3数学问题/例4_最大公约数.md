# 例4_最大公约数

**题目描述：**

```
输入两个数，求最大公约数
```

**输入：**

```
测试数据有多组，一组一行
```

**输出：**

```
一组一行
```

**输入样例：**

```
49 14
```

**输出样例：**

```
7
```



**思路：**

```
欧几里得算法
```



**代码：**

```c++
#include<stdio.h>

int gcd(int a, int b) {
	if (b == 0)return b;
	else return (b, a%b);
}

int main() {
	int a, b;
	while (scanf("%d%d", &a, &b) != EOF)
		printf("%d\n", gcd(a, b));
	return 0;
}
```
