# 例2_成绩排序

**题目描述：**

```
有N个学生的数据，将学生的数据按成绩排序，成绩相同则按姓名的字母顺序排序.
如还相同，则按年龄排序。小的在前。
```

**输入：**

```
测试数据有多组，每组第一行输入一个整数N(N <= 100)，接下来N行输入N个学生的信息。
每个学生的信息包括姓名（长度不超过100的字符串）、年龄（整数型）、成绩（小于100的整数）
```

**输出：**

```
将排好序后的学生成绩输出。格式：姓名 年龄 成绩
```

**输入样例：**

```
3
abc 20 99
bcd 19 97
bed 20 97
```

**输出样例：**

```
bcd 19 97
bed 20 19
abc 20 99
```



**思路：**

```
有多组测试数据，同上题处理。这里用运算符重载来处理结构体的比较.
关于strcmp函数：若str1=str2，返回零；str1<str2，则返回负数；str1>str2，则返回正数。
```



**代码：**

```c++
#include<iostream>
#include<algorithm>
#include<string>
using namespace std;

struct E{
	char name[101];
    int age;
    int score;
    bool operator < (const E &b) const{
		if(score != b.score) return score < b.score;
        int tmp = strcmp(name, b.name);
        if(tmp != 0) 
            return tmp < 0;
        else
            return age < b.age;
    }
}buf[1000];

int main(){
    int n;
    while(scanf("%d", &n) != EOF){
		for(int i = 0; i < n; i++)
            scanf("%s %d %d", buf[i].name, &buf[i].age, &buf[i].score);
        sort(buf, buf + n);
        for(int i = 0; i < n; i++)
            printf("%s %d %d\n", buf[i].name, buf[i].age, buf[i].score);
    }
    return 0;
}
```
