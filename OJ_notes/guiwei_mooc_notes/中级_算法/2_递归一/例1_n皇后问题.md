**题目描述：**

```
输入整数n，要求n个国际象棋的皇后，摆在n*n的棋盘上，互相不能攻击，输出全部方案。
输出结果里每一行代表一种摆法。行里的第i个数字是a，就代表第i行的皇后摆放在第a列。
编号从1开始。
```

**输入：**

```

```

**输出：**

```

```

**输入样例：**

```
4
```

**输出样例：**

```
2 4 1 3 
3 1 4 2
```



**思路：**

```
不知道是几个皇后，用递归代替循环。
```



**代码：**

```c++
#include<iostream>
#include<cmath>
using namespace std;
int N;
int queenPos[100];		//用来放皇后的位置，queenPos[i]表示第i行的皇后在第queenPos[i]列。

void NQueen(int k);

int main() {
	cin >> N;
	NQueen(0);		//从0行开始，程序中还是从0开始
	return 0;
}


//在0~k-1行皇后已经摆好的情况下，摆第k行及其后的皇后
void NQueen(int k) {
	int i;
	if (k == N) {		//N个皇后（0~k-1共k个）都已经摆好,则输出
		for (int i = 0; i < N; i++)
			cout << queenPos[i] + 1 << " ";		//输出是从1开始，所以加1
		cout << endl;
		return;
	}

	//逐个尝试第k个皇后的位置，i是第k行皇后的列号，j是行号
	for (i = 0; i < N; i++) {
		int j;
		for (j = 0; j < k; j++) {
			if (queenPos[j] == i || abs(queenPos[j] - i) == abs(k - j))		//列相等，或行列差相等（斜着）,则不能放
				break;
		}
		if (j == k) {	//当前选择位置不冲突
			queenPos[k] = i;
			NQueen(k + 1);
		}
	}
}
```
